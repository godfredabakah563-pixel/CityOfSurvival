# Installation Guide

## Step 1: Prerequisites  
Make sure you have the following installed on your system:
- Unity Hub
- Unity version 2020.3 or later
- Visual Studio or any other code editor you prefer

## Step 2: Clone the Repository  
Open your terminal and run the following command:  
```bash  
git clone https://github.com/godfredabakah563-pixel/CityOfSurvival.git
```  
Navigate into the project directory:  
```bash  
cd CityOfSurvival
```

## Step 3: Open the Project  
Launch Unity Hub and click on the "Add" button.  
Browse to the `CityOfSurvival` folder you just cloned and select it.  
Then, click on the project to open it.

## Step 4: Install Dependencies  
After the project is open, Unity may prompt you to install missing dependencies.  
Make sure to accept any prompts to install required packages.

## Step 5: Configure Project Settings  
Verify that the project settings are configured correctly for your platform.  
Go to `Edit > Project Settings` and check the following:
- Player settings  
- Input settings  

## Step 6: Build the Project  
To build the project, go to `File > Build Settings`.  
Select your target platform and click on the "Build" button.

## Step 7: Run the Project  
Once the build is complete, navigate to the build output directory and double-click the executable file to run the project.

## Troubleshooting  
If you encounter any issues, refer to the [Unity documentation](https://docs.unity3d.com/Manual/index.html) or reach out to the support community.

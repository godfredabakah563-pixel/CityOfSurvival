# City Of Survival

## Project Documentation

### Features
- Open-world exploration
- Crafting and building mechanics
- Survival elements (food, health, and shelter)
- Multiplayer mode
- Rich storyline and quests

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/godfredabakah563-pixel/CityOfSurvival.git
   ```
2. Navigate to the project directory:
   ```bash
   cd CityOfSurvival
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the game:
   ```bash
   npm start
   ```

### Controls
- **WASD**: Move
- **Space**: Jump
- **E**: Interact
- **I**: Inventory
- **M**: Map
- **Escape**: Pause

For more detailed information, please refer to the documentation in the repo and the help section in the game.

# City of Survival Quick Start Guide

Welcome to the City of Survival game! Follow this guide to kickstart your adventure in just 15 minutes.

## Prerequisites
- Ensure that you have installed the latest version of [Unity](https://unity.com/).
- Download and install [Visual Studio](https://visualstudio.microsoft.com/) for scripting.

## Quick Start Steps

1. **Clone the Repository**
   ```
   git clone https://github.com/godfredabakah563-pixel/CityOfSurvival.git
   cd CityOfSurvival
   ```

2. **Open the Project in Unity**
   - Open Unity Hub and click on `Add Project`.
   - Select the folder where you cloned the CityOfSurvival repository.

3. **Install Required Assets**
   - Unity will prompt you to install any necessary assets. Please ensure you do so.

4. **Set Up the Scene**
   - Open the `Main Scene` from the `Scenes` folder.
   - Press the `Play` button to test the initial gameplay.

5. **Modify Game Settings**
   - Access the `GameManager` in the `Hierarchy`.
   - Change parameters like spawn rates and difficulty settings before testing again.

6. **Build the Game**
   - Navigate to `File > Build Settings`.
   - Select your target platform and click `Build`. Follow the prompts to create your game build.

## Explore Further
After these quick steps, take your time to explore the different features and mechanics of the game. Check out the README for detailed documentation and advanced configurations.

Happy gaming!
